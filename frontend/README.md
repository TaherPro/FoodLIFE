FoodLIFE
Is an application designed to help communities manage food donations. 

The platform connects donors, recipients, and food bank staff in one organized system to reduce food waste and improve food distribution.

Main Users
- Staff
- Donor
- Recipient

Tech Stack
Frontend
- HTML 
- CSS
- JavaScript
- React
Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Auth


Main Features
- Full authentication system (JWT + role-based access)
- CRUD operations for donations and requests
- Staff dashboard for real-time management
- Recipient dashboard to browse and request food
- Donor dashboard to manage donations
- Simple and responsive React UI
- Secure REST API with protected routes
